1 Tutorials

1.1 Getting Started

Welcome to Genome Compiler! This tutorial will show you how to get started with using our software.

1.1.1 Creating an Account
  • If you do not have an account yet, sign up at designer.genomecompiler.com/ users/sign_up (Figure 1.1.1.1). You will receive an email welcoming you to the software. Activate your account by clicking the link provided in the email.


    Figure 1.1.1.1: Signing up to Genome Compiler.

1.1.2 Logging in
  • Once you have an account, simply login at designer.genomecompiler.com (Figure 1.1.2.1).


    Figure 1.1.2.1: Logging in to Genome Compiler.

  • You are now ready to go! Check out the rest of the manual to learn more about the different features of our software.

1.2 Genome Compiler User Interface

Genome Compiler offers a customizable user friendly interface.

1.2.1 Overview of Genome Compiler User Interface


Figure 1.2.1.1: General view of the software.

  • The toolbar at the top of the screen (Figure 1.2.1.2) contains icons for file operations, standard editing (see section 1.4) and tools.


    Figure 1.2.1.2: The toolbar.

  • The Materials box on the left of the screen (Figure 1.2.1.3) gives you access to all your files. The search tool helps you find your files easily (see section 1.6). You can also use it to search through the NCBI database (see section 1.8). The Materials box provides a number of icons for file operations and tools.


    Figure 1.2.1.3: The Materials box.

  • The properties section at the bottom of the screen (Figure 1.2.1.4) contains different tabs providing information on your project, parts, selection, compilation and restriction digests (more information on the properties section can be found in section 1.2.9).


    Figure 1.2.1.4: The properties section.

1.2.2 Tooltips
  • Many tooltips are available, describing dialogs, icons, etc. Simply hover over the item to see the tooltip. For example, information regarding the amino acids is displayed when you hover over them (Figure 1.2.2.1).


    Figure 1.2.2.1: The tooltip describing the amino acids.

1.2.3 Minimizing Materials box and Properties
  • The Materials box and the properties section can be minimized by clicking the appropriate icon (Figures 1.2.3.1 and 1.2.3.2).


    Figure 1.2.3.1: Minimizing buttons for the Materials box and the properties section.


    Figure 1.2.3.2: Minimized Materials box and properties section.

1.2.4 Maximizing Materials box and Properties
  • The Materials box and the properties section sizes can be adjusted (Figure 1.2.4.1 and 1.2.4.2).


    Figure 1.2.4.1: Buttons to adjust the size of the Materials box and the properties section.


    Figure 1.2.4.2: Enlarged Materials box and properties section.

1.2.5 Re-arranging Project Tabs
  • Project tabs can be moved and organized. When multiple projects are open, they can appear in two different views. In tile view, you can adjust the size of each project’s pane by moving the bar in between them (Figure 1.2.5.1).


    Figure 1.2.5.1: Tile view with multiple projects.

  • Alternatively, in full view, you can have only one project visible at a time. Navigate to each project by clicking the tab (Figure 1.2.5.2).


    Figure 1.2.5.2: Full view with multiple projects.

  • Navigate between both views by moving the tabs or by clicking the appropriate icon (Figure 1.2.5.4).

    The swith to full view icon.  

    The switch to tile view icon.

    Figure 1.2.5.4: Switch view icons.

1.2.6 Viewing Projects
  • You can choose between two different views to visualize projects. The split view shows DNA on the left pane and the circular or linear view on the right pane (Figure 1.2.6.1). Alternatively, the expanded view allows you to select one view which fills the whole screen (Figure 1.2.6.2).


    Figure 1.2.6.1: The split view.


    Figure 1.2.6.2: The expanded view.

  • Navigate between the views using the ”Expand View” and ”Split View” buttons (Figure 1.2.6.4).

    The ”Expand View” button.  

    The ”Split View” button.

    Figure 1.2.6.4: Switch view icons.

1.2.7 Abstraction Layers
  • There are three abstraction layers: the circular view, the linear view and the DNA view. You can navigate to each layer view by clicking its icon (Figures 1.2.7.1 and 1.2.7.2).


    Figure 1.2.7.1: The linear view and its icon.


    Figure 1.2.7.2: The DNA view and its icon.

  • The DNA view contains an amino acid layer (Figure 1.2.7.3). Methionine is shown in green, stop codons are represented by a red star, and incomplete amino acids are marked by a red exclamation point.


    Figure 1.2.7.3: The amino acid layer in the DNA view.

  • You can show or unshow the amino acid layer by checking or unchecking the box in the toolbar drop down Layers menu (Figure 1.2.7.4).


    Figure 1.2.7.4: The ”Amino Acid” checkbox in the toolbar drop down Layers menu.

1.2.8 Numbering of Base Pairs and Amino Acids
  • Base pairs and amino acids are numbered in the DNA view (Figure 1.2.8.1). After each thousand, counting starts again from 0 and a label indicates the current thousand (Figure 1.2.8.2).


    Figure 1.2.8.1: Numbered base pairs and amino acids in the DNA view.

  • After each thousand, counting starts again for base pairs from 0 and a label indicates the current thousand (Figure 1.2.8.2).


    Figure 1.2.8.2: Numbered base pairs in the DNA view, after a thousand.

  • The number of the first base pair of each part is indicated in the linear view (Figure 1.2.8.3).


    Figure 1.2.8.3: The number of the first base pair of each part in the linear view.

1.2.9 Properties Section
  • The properties section at the bottom of the screen contains different tabs providing information on your project (Figure 1.2.9.1), parts (Figure 1.2.9.2) (the tab is labeled ”Multiple” when multiple parts are selected), selection (Figure 1.2.9.5), compilation (Figure 1.2.9.3) and restriction digests (Figure 1.2.9.4).


    Figure 1.2.9.1: The project tab.


    Figure 1.2.9.2: The part tab.


    Figure 1.2.9.3: The compilation tab.


    Figure 1.2.9.4: The digest tab.

  • When you select a strand of DNA in the DNA view, the selection tab (Figure 1.2.9.5) displays:
    • the selection count
    • the GC content
    • the melting temperature (calculated according to SantaLucia 1998)
    • the start of the selection
    • the end of the selection
    • the number of parts contained in the selection
    • the isolelectric point
    • the hydrophobicity
    • the molecular weight

    as well as the name, linear formula and type of the amino acid, when a single amino acid is selected.


    Figure 1.2.9.5: The selection tab.

1.2.10 Options Menu
  • The ”Options” drop down menu at the top right (Figure 1.2.10.1) allows you to find information about the software, to give feedback and to access your preferences.


    Figure 1.2.10.1: The ”Options” drop down menu.

  • By default, projects left open in previous sessions are reopened in new sessions. This can be changed by selecting ”No” in the ”Preferences” dialog (Figure 1.2.10.2). You can also reset all dialogs by clicking ”Reset all dialogs”.


    Figure 1.2.10.2: The ”Preferences” dialog.

1.3 Importing and Exporting

Genome Compiler supports import of Genome Compiler (.gcproj), INSD (.xml), GenBank (.gb), FASTA (.fasta, .fas, .fa), Raw DNA (.txt), Geneious (.geneious) and Vector NTI archive (.ma4) file formats. Genome Compiler supports the export of Genome Compiler, GenBank, FASTA and Raw DNA file formats.

1.3.1 Importing Files
  • To import a file, click the “Import File” icon in the Materials box (Figure 1.3.1.1).


    Figure 1.3.1.1: The ”Import File” icon in the Materials box.

  • A dialog will pop up (Figure 1.3.1.2) to allow you to choose the file from your computer files.


    Figure 1.3.1.2: Choosing a file from your computer files.

  • Once you selected a file, a dialog will pop up (Figure 1.3.1.3) to prompt you for a location. You can choose to save the file in an existing folder or to create a new folder.


    Figure 1.3.1.3: The ”Save on Cloud” dialog when importing a file.

  • To create a new folder, click “New Folder”. A dialog will pop up (Figure 1.3.1.4) to prompt you for a name and an optional folder description. Save your file by clicking “Save”. The file will now be saved as a new project in Genome Compiler and can be opened from the Materials box.


    Figure 1.3.1.4: The ”Save on Cloud” dialog when creating a new folder.

1.3.2 Importing Parts
  • To import a part, select “Parts” from the import drop down menu (Figure 1.3.2.1).


    Figure 1.3.2.1: The import drop down menu in the Materials box.

  • The “Import Genetic Parts” dialog will appear (Figure 1.3.2.2). The “Create New Part” icon allows you to add a new part to the list. Select the appropriate feature from the “Feature Key” drop down menu and enter the part name and sequence. Click the “Delete” icon to delete a part.


    Figure 1.3.2.2: The ”Import Genetic Parts” dialog and its ”Feature Key” drop down menu.

  • Import the listed parts by clicking “Import”. The flow for saving parts is the same for saving files. Just like imported files, the imported part is saved as a new project in the Materials box.

1.3.3 Exporting Files
  • To export a file, select it and select the appropriate file format from the “Export” drop down menu in the Materials box (Figures 1.3.3.1) or in the toolbar (Figures 1.3.3.2). A dialog will pop up to prompt you for a name and a location.


    Figure 1.3.3.1: The “Export” drop down menu in the Materials box.


    Figure 1.3.3.2: The “Export” drop down menu in the toolbar.

1.4 Editing

All the standard editing tools are available in the software. You can copy, cut, paste, delete, drag, drop, undo and redo, as well as perform reverse complements, create, merge and split parts or manage DNA strands.

1.4.1 Basic Copy/Cut/Paste/Delete/Undo/Redo
  • You can copy and paste, cut and paste or delete parts, amino acids or base pairs within a project or in between different projects. Right click on the sequence and select from the drop down menu (Figure 1.4.1.1), or use the icons in the toolbar (Figure 1.4.1.2) or the standard keyboard shortcuts. You can paste strands of DNA from external sources.


    Figure 1.4.1.1: Cut, copy, paste and delete in the right click drop down menu.


    Figure 1.4.1.2: Cut, copy, paste and delete icons in the toolbar.

  • You can undo or redo changes via the icons in the toolbar (Figure 1.4.1.3) or the standard keyboard shortcuts.


    Figure 1.4.1.3: Undo and redo icons in the toolbar.

1.4.2 Drag and Drop Functionality
  • In addition to standard copy, cut, paste, delete, Genome Compiler also has drag and drop functionality.
  • You can drag and drop projects from the Materials box to open projects, or from open projects to other open projects.
  • You can also drag and drop parts, amino acids or base pairs within a project and between projects.
  • You can create a new project by dragging a selection to the toolbar (Figures 1.4.2.1 and 1.4.2.2).


    Figure 1.4.2.1: Drag and drop a selection to the toolbar to create a new project.


    Figure 1.4.2.2: New project created from a dropped selection.

1.4.3 Adding New Parts
  • You can add new parts to your projects. Right click where you would like your new part to be and select “New part” from the drop down menu (Figure 1.4.3.1), or click the “Create New Part” icon in the toolbar (Figure 1.4.3.2).


    Figure 1.4.3.1: ”New part” in the right click drop down menu.


    Figure 1.4.3.2: The ”New part” icon in the toolbar.

  • This will create an empty part (Figure 1.4.3.3) which can be edited in the DNA view (Figure 1.4.3.4).


    Figure 1.4.3.3: New empty part in the linear view.


    Figure 1.4.3.4: New empty part in the DNA view.

1.4.4 Adding Amino Acids
  • You can add amino acids in your projects. Right click on the amino acid after which you would like the amino acids to be added and select “Insert Amino Acids...” from the drop down menu (Figure 1.4.4.1).


    Figure 1.4.4.1: ”Insert Amino Acids” in the right click drop down menu.

  • The ”Insert Amino Acid” dialog will appear. Type in the amino acids (use the 1 letter representation).
  • The ”Translation Settings” dialog will appear. You can choose to translate according to codon usage or to a translation table. For more details about translation see section 1.14.

1.4.5 Reverse Complement
  • To perform a reverse complement, right click on the part and select “Reverse complement” from the drop down menu (Figure 1.4.5.1), or click the “Reverse Complement” icon in the toolbar (Figure 1.4.5.2).


    Figure 1.4.5.1: ”Reverse complement” in the right click drop down menu.


    Figure 1.4.5.2: The ”Reverse Complement” icon in the toolbar.

1.4.6 Merging and Splitting Parts
  • You can merge and split parts. To merge parts, select them and select “Merge” in the toolbar (Figure 1.4.6.1), or right click and select “Merge selected parts” from the drop down menu (Figure 1.4.6.2). A new merged part containing the selection will be created.


    Figure 1.4.6.1: The ”Merge” icon in the toolbar.


    Figure 1.4.6.2: ”Merge selected parts” in the right click drop down menu.

  • To split a sequence, switch to DNA view and place the caret where you would like to split the sequence. Select “Split” in the toolbar (Figure 1.4.6.3), or right click and select “Split part” from the dropdown menu (Figure 1.4.6.4). New split parts will be created.


    Figure 1.4.6.3: The ”Split” icon in the toolbar.


    Figure 1.4.6.4: ”Split part” in the right click drop down menu.

1.4.7 Selecting Strand
  • You can select DNA strands using the ”Minus strand” and ”Plus strand” icons in the ”Part” tab (Figure 1.4.7.1).


    Figure 1.4.7.1: The ”Minus strand” and ”Plus strand” icons in the ”Part” tab.

  • The selected strand will be indicated by a darker color, as well as identified by an arrow (Figure 1.4.7.3).

    Plus strand selected.  

    Minus strand selected.

    Figure 1.4.7.3: DNA strands visualization.

  • You can show or unshow the minus strand by checking or unchecking the box in the toolbar drop down Layers menu (Figure 1.4.7.4).


    Figure 1.4.7.4: The ”Minus Strand” checkbox in the toolbar drop down Layers menu.

1.4.8 Mutation Wheel
  • The mutation wheel allows you to mutate amino acids. To access the mutation wheel, click the drop down arrow below the amino acid and select ”Mutate...” (Figure 1.4.8.1) or right click on the amino acid and select ”Mutate...” from the drop down menu (Figure 1.4.8.2).


    Figure 1.4.8.1: ”Mutate...” in the drop down arrow menu.


    Figure 1.4.8.2: ”Mutate” in the right click drop down menu.

  • Select amino acids from the mutation wheel (Figure 1.4.8.3) and click the green tick. You can filter selection of amino acids with a specific property by checking the property checkbox on the left.


    Figure 1.4.8.3: The amino acids mutation wheel.

1.4.9 Basic Local Alignment Search Tool (BLAST)
  • You can send sequences to Basic Local Alignment Search Tool (BLAST) directly from the software. Select a sequence in the DNA view or a part in the linear view, right click and select ”Send to BLAST” from the drop down menu (Figure 1.4.9.1).


    Figure 1.4.9.1: ”Send to BLAST” in the right click drop down menu.

  • This will open the BLAST website (blast.ncbi.nlm.nih.gov) with the selected sequence pasted.

  • The search tool will look for your keyword in everything contained in the project meta data, including its name, comments, etc. Meta data for each project can be found in the project tab (Figure 1.6.1.2) located along the bottom of each opened project or in a tooltip while hovering over the closed project in the Materials box (Figure 1.6.1.3).


    Figure 1.6.1.2: The project tab showing the project meta data.


    Figure 1.6.1.3: Tooltip showing the project meta data.

    • 1.6.2 Example
    • For example, you can find the genome of this strand of E.coli by looking for the GI number that appears in the comment section (Figure 1.6.2.1).


      Figure 1.6.2.1: Example: search for the GI number.

    1.7 Search in Projects

    The search bar allows you to search for DNA sequences and for part meta data within your projects.

    1.7.1 Searching in Projects
    • Type a keyword or a DNA sequence in the search bar (Figure 1.7.1.1) in the top right of the screen and click enter. The arrows allow you to navigate through the results.


      Figure 1.7.1.1: The search bar in the top right of the screen.

    • The search tool will look for your keyword in everything contained in the part meta data, including its name, comments, etc. Meta data for each part can be found in the part tab (Figure 1.7.1.2) located along the bottom of each opened project or in a tooltip while hovering over the part (Figure 1.7.1.3). For example in this case, the keyword ”transposase” was found in the “Product” field.


      Figure 1.7.1.2: Part tab showing the part meta data.


      Figure 1.7.1.3: Tooltip showing the part meta data.

    • DNA sequences are searched for on both strands (Figure 1.7.1.4).


      Figure 1.7.1.4: Search for DNA sequences using the toolbar.

    1.8 NCBI Search

    The NCBI search tool allows you to search through the NCBI database directly from our software.

    1.8.1 Searching in the NCBI Database
    • Type a keyword in the search bar and click the NCBI icon (Figure 1.8.1.1).


      Figure 1.8.1.1: The NCBI icon and the search bar in the Materials box.

    • NCBI search results will appear (Figure 1.8.1.2). The icon on the right of each result takes you to the NCBI website. To download the file, click the green “import” icon. To close the NCBI search results, click the NCBI icon.


      Figure 1.8.1.2: NCBI search results, with - from left to right - the NCBI icon, the green ”import” icon and the ”See item in the NCBI website” icon.

    • The downloaded file will be automatically added to the cloud in the Materials box (Figure 1.8.1.3).


      Figure 1.8.1.3: The downloaded file in the Materials box.

    • If you have already found what you are looking for in the NCBI database, you can copy the name or the GI number, and paste it in the search bar (Figure 1.8.1.1). Click the NCBI icon, and download the file.

    1.8.2 Downloaded and Downloading Files Window
    • Downloaded and downloading files appear in a window at the bottom (Figure 1.8.2.1). You can remove files from the list by clicking the “remove” icon. The “view in NCBI” icon takes you to the corresponding page on the NCBI website. Click the “view in the Materials box” icon on the left of the downloaded project to view the file in the Materials box.


      Figure 1.8.2.1: The downloaded and downloading files window.

    1.9 Sharing and Collaboration

    The Cloud allows users to access projects from anywhere and share them with others.

    1.9.1 Cloud Folders
    • Folders saved in the cloud appear in the Materials box and are marked with a cloud icon. Refresh the cloud by clicking the “Refresh Cloud” icon (Figure 1.9.1.1).


      Figure 1.9.1.1: The ”Refresh Cloud” icon in the Materials box.

    1.9.2 Creating New Folders
    • To create a new folder, right click and select “New Cloud Folder” (Figure 1.9.2.1). You can also create new folders via the ”Cloud Folder Settings” dialog as explained in section 1.9.5.


      Figure 1.9.2.1: ”New Cloud Folder” in the right click menu.

    1.9.3 Creating Subfolders
    • You can also create subfolders. Click the “New Folder” icon in the Materials box, or right click and select “New Folder” (Figure 1.9.3.2).

      The ”New Folder” icon in the Materials box.  

      ”New Folder” in the right click drop down menu.

      Figure 1.9.3.2: Create Cloud subfolders in the Materials box.

    1.9.4 Sharing Cloud Folders
    • Folders can be shared via the “Cloud Folder Settings” dialog. It can be accessed by clicking the “Cloud Settings” icon in the Materials box or in the toolbar (Figure 1.9.4.2). In this dialog select the folder from the drop down menu which you would like to share.

      The ”Cloud Settings” icon in the Materials box.  

      The ”Cloud Settings” icon in the toolbar.

      Figure 1.9.4.2: The ”Cloud Settings” icon.

    • You can also share a folder directly by right clicking on a folder and selecting “Settings” (Figure 1.9.4.3). Please note that you cannot share subfolders.


      Figure 1.9.4.3: ”Settings” in the right click drop down menu.

    • You can share folders with other users. Select which folder you would like to share from the drop down menu (Figure 1.9.4.4). Enter their email address in the ”Share with” field and click “Add”. They will receive an email telling them that you would like to share a folder.


      Figure 1.9.4.4: The ”Cloud Folder Settings” dialog: choosing a folder.

    • To manage permissions, select “Read/Write” or “Read” from the drop down menu (Figure 1.9.4.5). To remove users from the list, click “Remove”.


      Figure 1.9.4.5: The ”Cloud Folder Settings” dialog: managing permissions.

    1.9.5 Creating New Folders via the ”Cloud Folder Settings” dialog
    • You can also create new folders from the “Cloud Folder Settings” dialog (Figure 1.9.5.1) by clicking on the “New Folder” icon. Enter the new folder’s name and an optional folder description.


      Figure 1.9.5.1: The ”New Folder” icon in the ”Cloud Folder Settings” dialog.

    1.10 User Comments

    The user comments feature allows you to make comments on all abstraction layers, view comments from multiple users over the cloud manipulating a single sequence, and to view a summary table of all comments in each project.
